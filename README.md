# 知网插件功能说明

## 功能介绍
1. **添加“下载 PDF”按钮**：在知网文章列表页面直接添加“下载 PDF”按钮，方便快速下载 PDF 格式文章。
2. **悬停显示摘要**：鼠标悬停在文章标题上时，自动显示论文或报纸的摘要。
3. **一键下载所有 PDF**：在下载页面添加“下载所有 PDF”按钮，支持一次性下载当前页面的所有 PDF 文件。

## 安装步骤
1. 打开浏览器（Edge 或 Chrome）。
2. 进入浏览器设置。
3. 点击“扩展程序”。
4. 开启“开发人员模式”。
5. 点击“加载解压缩的扩展”。
6. 选择插件文件夹。
